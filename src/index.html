Hydraulic Winch System — Core Equations (Unitless Form)

Symbols (reference units only)

Symbol Description Units

Dp pump displacement cc/rev
Np number of pumps –
ηvp pump volumetric efficiency –
Qp,i actual per-pump flow cc/min
Qtot total manifold flow cc/min
Nm number of motors in parallel –
Dm motor displacement cc/rev
ηvm motor volumetric efficiency –
nm motor speed rpm
nd drum (ring) speed rpm
Ggb gearbox reduction –
Gpr pinion→ring reduction –
Gtot total reduction = GgbGpr –
Dℓ(z) effective layer diameter at depth z m
vflow(z) flow-limited line speed m/s
Pnom total nominal electric motor power kW
ηem electromechanical efficiency –
Pavail available shaft power kW
T(z) operating line tension N
vpwr(z) power-limited line speed m/s
vavail(z) available line speed m/s
Wpay,sub payload weight in water N
wc,sub cable weight in water per unit length N/m
L max usable cable length m
Tstatic(z) static line tension N
ΔTa, ΔTd, ΔTh dynamic tension adders (accel, drag, heave) N
τm motor torque N·m
ηtrain drivetrain efficiency –
ηmm mechanical efficiency of motor –
rℓ(z) effective layer radius m
rcore core radius m
pc cable effective pitch (≈ cable dia × packing factor) m

A) Available Line Speed — Unitless Relations

  1. Qp,i / (ne Dp) = ηvp
  2. Qtot / (Np Qp,i) = 1
  3. Qm / (Qtot / Nm) = 1
  4. nm Dm / (ηvm Qm) = 1
  5. nd / (nm / Gtot) = 1, where Gtot = GgbGpr
  6. vflow(z) / ((π Dℓ(z) / 60) × nd) = 1
  7. Pavail / (ηem Pnom) = 1
  8. vpwr(z) T(z) / Pavail = 1
  9. vavail(z) = min(vflow(z), vpwr(z))

B) Tension vs Depth — Unitless Relations

  1. (Tstatic(z) - Wpay,sub) / (wc,sub z) = 1 for 0 ≤ z ≤ L
  2. (Tstatic(z) - Wpay,sub) / (wc,sub L) = 1 for z > L
  3. (T(z) - Tstatic(z)) / (ΔTa + ΔTd + ΔTh) = 1
  4. T(z) / (Tstatic(z) + ΔTa + ΔTd + ΔTh) = 1

C) Cross-Checks (Optional)

  1. Tcap(z) rℓ(z) Gtot / (τm Nm ηtrain) = 1
  2. τm / ((Δp Dm ηmm) / (2π)) = 1
  3. Dℓ(i) / (2 rcore + 2 (i - 0.5) pc) = 1

Each equation expresses the physical dependency as a pure ratio, enabling unit-consistent implementation regardless of the measurement system. All efficiencies, ratios, and reductions are dimensionless. Power-, flow-, and tension-limited speeds follow directly from these relationships.

How to update component presets

The component dropdowns now read from dedicated catalogs in src/js/component-selectors.mjs. Add or edit part numbers directly in the matching array to make them available across the app.

  1. Open src/js/component-selectors.mjs and locate the array that matches the component type you want to change (for example GEARBOX_OPTIONS or ELECTRIC_MOTOR_OPTIONS).
  2. Add a new object with the part number, description, and any field values that should autofill when the part is selected. Follow the existing entries as a template.
  3. Save the file and refresh the page. The new entry will appear under the corresponding dropdown and auto-populate the linked form inputs.

Tip: if you need to clear a selection after experimenting, change any populated value and the app will revert the dropdown to Custom (manual input) automatically.
